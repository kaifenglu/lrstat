# Print Multi-Arm Multi-Stage Design

Prints the stopping boundaries and power for multi-arm multi-stage
design.

## Usage

``` r
# S3 method for class 'mams'
print(x, ...)
```

## Arguments

- x:

  The design object to print.

- ...:

  Ensures that all arguments starting from "..." are named.

## Value

A tabular printout of the design elements.
