# Print Two-Stage Seamless Sequential Design

Prints the stopping boundaries and power for two-stage seamless
sequential design.

## Usage

``` r
# S3 method for class 'seamless'
print(x, ...)
```

## Arguments

- x:

  The design object to print.

- ...:

  Ensures that all arguments starting from "..." are named.

## Value

A tabular printout of the design elements.
